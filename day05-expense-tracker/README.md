# Day 05 - Expense Tracker

A Python script to track daily expenses stored in a CSV file.

## Features
- Add expenses with category, amount and description
- View all expenses in a formatted table
- Filter expenses by category
- View summary report by category
- Data stored in CSV — no database needed

## Usage
```bash
python tracker.py
```

## Categories
Food, Transport, Shopping, Bills, Health, Entertainment, Others

## Example
```
💰 Expense Tracker
------------------------------
1. Add Expense
2. View All Expenses
3. View by Category
4. View Summary
5. Exit

Choose option (1-5): 1
Category: Food
Amount (₹): 150
Description: Lunch
✅ Expense added: Food - ₹150
```
